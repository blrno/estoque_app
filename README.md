# inciando a aplicação
- clone o projeto no seu computador
- crie um ambiente virtual python venv na raiz do projeto
- inicie o ambiente virtual activate
- instale a dependencia django utilizando pip install
- rode os comandos de makemigrations e migrate para criar o bd
- rode o comando createsuperuser 
- rode a aplicação com o comando do django runserver
- acesse a url fornecida pelo django
